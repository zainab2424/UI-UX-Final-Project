# Pixel Quest - UX Final Project

A complete multi-page website built from Figma design, featuring Home, Track, and Rewards pages. Built with React + TailwindCSS, recreating all layout spacing and colors from the original design.

## Features

- **Home Page**: Hero section, About Us, Testimonials, Clients section
- **Track Page**: Habit Tracker, Mood Tracker, Note Taker, Pomodoro Timer, To Do List
- **Rewards Page**: Challenges, Progress Tracking, Badges, Level System
- **Navigation**: Responsive navbar and footer with routing
- **Design**: Faithful recreation of Figma design with exact colors and spacing

## Tech Stack

- React 18
- Vite
- TailwindCSS
- React Router DOM
